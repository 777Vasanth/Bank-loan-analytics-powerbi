# 🏦 Bank Loan Analytics Dashboard (Power BI)

🚀 End-to-end Power BI dashboard analyzing loan performance, default risk, and customer segmentation in a banking environment

## 📌 Overview

This project presents a comprehensive Loan Analytics Dashboard built using Power BI to monitor loan disbursement, track default risk, and analyze customer and branch performance.

The dashboard enables financial institutions to identify high-risk segments, optimize lending strategies, and improve portfolio performance.

---

## 🎯 Key Metrics

* Total Loans: 79K
* Total Amount Disbursed: ₹86.51Bn
* Default Rate: ~12%
* Total Overdue Amount: ₹2Bn

---

## 🔍 Key Insights

* Home Loans and Business Loans contribute the highest loan volume 
* Default rates are relatively consistent across loan types (~11–12%), indicating systemic risk patterns
* Customers with lower income groups show slightly higher default rates
* High overdue amounts are concentrated among specific high-risk customers
* Certain branches outperform others significantly in loan disbursement
* Credit score segmentation shows minimal variation in default rate, suggesting other factors influence risk
